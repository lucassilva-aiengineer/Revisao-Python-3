Objetivos do projeto utilizar os conceitos de programação: POO, funções, funções especiais, funções lambda, loops for, loops while,
estruturas de condicionais, lists compressions, exceções, estruturas de dados. 


Título: Um sistema de mercado inteligente. 


Descrição: Um sistema de mercado formado por três componentes prinípais. Uma interface voltada ao cliente,  uma interface voltada ao controle de estoque, uma interface voltada a administração dos funcionários.
